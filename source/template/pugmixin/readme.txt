директория для миксинов pug

******************************************************************
_svgicon-mixin.pug миксин для быстрого встраивания svg иконок
******************************************************************

использование в разметке:

+icon('название иконки из директории с иконками svg')

либо с модификатором если он дополнительно определен:

+icon('arr_round','red_mod')

.icon-arr_round {
    font-size: 10rem;
    &.red_mod {
        color:red;
    }
}

******************************************************************
